### People Tracking 

Mainly, here I used several approaches for people tracking task.
I've tried these approaches\tecnhiques:
- MobileNet v2
- SSD (COCO)
- SSD(Custom people dataset)
- OpenCV trackers
- SORT 
- Dlib tracker
- ReIdentification.

Technical stack:
- Python3 
- Docker
- Jupyter Notebook
- Tensorflow
- OpenVINO

We final solution works with ~20 fps on MacBook Pro
